How I Work
Three operating principles — consistent across every environment
⚙️
Operate
I run complex, multi-stakeholder workflows with precision. Ten accounts. Competing deadlines. Unclear ownership. I create structure where there isn't any, and maintain it under pressure. Things don't fall through because I build the systems that catch them.
🏗️
Build
I create infrastructure before it's asked for. Notion databases. Excel trackers with automated error-flagging. QA frameworks. HTML briefing systems. Process playbooks. Everything built to outlast my presence — documented for the next person from day one.
👥
Develop
I train, evaluate, and give direct feedback. Structured programs with scored evaluations across five dimensions. Clear readiness ratings. Written developmental feedback that doesn't hedge. I document so the next person doesn't start from scratch.
